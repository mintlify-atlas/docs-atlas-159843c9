## Endpoint

```
GET /api/animes/full
```

## Description

Fetch a complete list of animes with extensive filtering capabilities. This endpoint returns comprehensive anime data including metadata, genres, studios, ratings, and more. Results are cached for 2 hours to optimize performance.

## Authentication

<Note>
This endpoint is publicly accessible and does not require authentication.
</Note>

## Rate Limiting

- **Limit**: 100 requests per 60 seconds per IP
- **Headers**: Rate limit information included in response headers

## Query Parameters

### Filtering Parameters

<ParamField query="genre_filter" type="string">
  Filter by anime genres. Supports multiple values.
  
  **Available genres**: Action, Adventure, Avant Garde, Award Winning, Boys Love, Comedy, Drama, Ecchi, Erotica, Fantasy, Girls Love, Gourmet, Horror, Mystery, Romance, Sci-Fi, Slice of Life, Sports, Supernatural, Suspense
  
  **Example**: `genre_filter=Action,Adventure`
</ParamField>

<ParamField query="type_filter" type="string">
  Filter by anime type.
  
  **Available types**: TV, Movie, OVA, ONA, Special, Music, CM, PV, TV Special
  
  **Example**: `type_filter=TV`
</ParamField>

<ParamField query="studio_filter" type="string">
  Filter by animation studio name. Supports multiple studios.
  
  **Example**: `studio_filter=MAPPA,Ufotable`
</ParamField>

<ParamField query="score_filter" type="number">
  Filter by minimum score (0-10).
  
  **Example**: `score_filter=8.5`
</ParamField>

<ParamField query="status_filter" type="string">
  Filter by airing status.
  
  **Available statuses**: Finished Airing, Currently Airing, Not yet aired
  
  **Example**: `status_filter=Currently Airing`
</ParamField>

<ParamField query="year_filter" type="number">
  Filter by release year.
  
  **Example**: `year_filter=2024`
</ParamField>

<ParamField query="rating_filter" type="string">
  Filter by age rating.
  
  **Available ratings**: 
  - `g - all ages` (E)
  - `pg - children` (E10+)
  - `pg-13 - teens 13 or older` (13+)
  - `r - 17+ (violence & profanity)` (15+)
  - `r+ - mild nudity` (17+)
  - `rx+-+hentai` (18+)
  
  **Example**: `rating_filter=pg-13 - teens 13 or older`
</ParamField>

<ParamField query="season_filter" type="string">
  Filter by anime season.
  
  **Available seasons**: Winter, Spring, Summer, Fall
  
  **Example**: `season_filter=Spring`
</ParamField>

<ParamField query="aired_day_filter" type="string">
  Filter by broadcast day of the week.
  
  **Available days**: Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday
  
  **Example**: `aired_day_filter=Saturday`
</ParamField>

<ParamField query="banners_filter" type="boolean">
  Filter to only show animes with banner images.
  
  **Example**: `banners_filter=true`
</ParamField>

### Sorting Parameters

<ParamField query="order_by" type="string">
  Sort results by specified field.
  
  **Available options**:
  - `score` - Sort by score (descending)
  - `score_asc` - Sort by score (ascending)
  - `title` - Sort by title (descending)
  - `title_asc` - Sort by title (ascending)
  
  **Example**: `order_by=score`
</ParamField>

### Content Control

<ParamField query="parental_control" type="boolean" default="true">
  Apply parental control filtering to exclude mature content.
  
  **Example**: `parental_control=false`
</ParamField>

<ParamField query="search_query" type="string">
  Search query to filter animes by title.
  
  **Example**: `search_query=attack on titan`
</ParamField>

## Examples

### Basic Request

<CodeGroup>
```bash cURL
curl -X GET "https://your-domain.com/api/animes/full"
```

```javascript JavaScript

const data = await response.json();

```

    "type": "rate_limit",
    "statusCode": 429
  }

<Note>
**Performance Tip**: This endpoint implements aggressive caching. Results are cached for 2 hours with both browser and CDN caching enabled.
</Note>

</Warning>
