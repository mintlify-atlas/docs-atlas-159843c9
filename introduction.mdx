## What is AniDev?

AniDev is a full-stack anime streaming platform built with modern web technologies. It provides a seamless experience for discovering, watching, and managing your anime collection with advanced features like personalized collections, progress tracking, and intelligent search.

## Technology Stack

AniDev is built on a modern, performant technology stack:

- **Frontend**: Astro v5.16.8 with React v19.2.3 for interactive components
- **Styling**: TailwindCSS v4.1.10 for responsive, beautiful designs
- **Backend**: Node.js with Astro server-side rendering
- **Database**: Supabase for authentication, storage, and real-time data
- **Caching**: Redis for high-performance API responses
- **Media**: Vidstack for streaming, Sharp for image optimization
- **State Management**: Zustand for client-side state

<Note>
AniDev uses TypeScript throughout the codebase, providing type safety and excellent developer experience.
</Note>
